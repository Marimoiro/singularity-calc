# 特異点 到達時間計算機 — Revolution Idle

[Revolution Idle](https://wikiwiki.jp/revoidle/) の **特異点(Singularity)** で、アトムのペナルティを考慮した「実際にかかる時間」を計算するツールです。

👉 **公開ページ:** https://marimoiro.github.io/singularity-calc/

## これは何？

アトムは毎秒 `a` 倍で乗算的に増えるため、単純には

```
t = log10(目標アトム / 現在アトム) ÷ log10(a)
```

で到達時間が求まります。しかし実際は、**アトム量が閾値を超えるたびに「原子獲得量」が累乗ペナルティを受け**、実効レートがどんどん下がります（例: 1e10 で `^0.5`、1e308 で `^0.12`、1e500 で `^0.05`）。

このツールはそのペナルティ区間ごとに実効レートを計算し、区間ごとの所要時間を積み上げることで「実際にかかる時間」を推定します。

## 使い方

1. **現在のアトム量**（例 `1.8e308`）と、アトム表示の下にある**毎秒獲得倍率**（例 `×1.013/秒`）を入力
2. **目標**を「特異点の回数」または「アトム量」で指定
3. 推定時間とペナルティ区間ごとの内訳が表示されます

入力した毎秒獲得倍率を現在アトム量でのペナルティ適用後の実効値とみなし、ペナルティ無しのベース倍率 `B` を逆算して各区間を計算しています。

## 前提・制限

- 特異点倍率や元素などのベース獲得量は周回中一定と仮定
- ツリー/精錬による各ペナルティの弱化、サクリファイスダスト・拡散力系の強化は未反映（安全側の見積り）
- 必要アトム量は wiki 記載の区間式に基づく近似値
