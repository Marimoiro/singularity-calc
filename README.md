# 特異点 到達時間計算機 — Revolution Idle

[Revolution Idle](https://store.steampowered.com/app/2763740/Revolution_Idle/) の **特異点(Singularity)** で、アトムのペナルティ・軽減・バフを考慮した「実際にかかる時間」を計算するツールです。日本語 / English 対応。

👉 **公開ページ:** https://marimoiro.github.io/singularity-calc/

## これは何？

アトムは毎秒 ×a になるので、**1秒で `log10(a)` 桁**増えます。到達時間は

```
t = log10(目標アトム / 現在アトム) ÷ log10(a)
```

ただし実際は、**アトム量が閾値を超えるたびに「原子獲得量」が累乗ペナルティ/バフを受け**、実効レートが変化します（1e10 で `^0.5`、1e308 で `^0.12`、1e500 で `^0.05` …）。指数はすべて掛け算で効くため、実効レートは `a = B^E`（B＝特異点倍率、E＝有効な全指数の積）になります。

このツールはペナルティ区間ごとに実効レートを計算し、区間ごとの所要時間を積み上げて「実際にかかる時間」を推定します。

## 計算の詳細

### 基本モデル
アトムは毎秒 ×a になる（乗算的に増える）ので、log10 で見ると **毎秒 `log10(a)` 桁** 増えます。現在アトム `A0` から目標 `A1` までの時間は:

```
t = ( log10(A1) − log10(A0) ) / log10(a)   [秒]
```

ざっくり言うと「倍率を10倍にすると増える桁/秒が +1」になります（1桁/秒 増やすには倍率×10）。

### 実効レート a = B^E
- `B` = 特異点倍率（ペナルティ未発動時の毎秒倍率）
- 「原子獲得量」は閾値を超えるたびに累乗され、これらの指数は **すべて掛け算** で効きます:

```
a = ((B^p1)^p2)^p3… = B^(p1·p2·p3·…) = B^E ,   E = Π pi
log10(a) = E · log10(B)
```

### 原子獲得量に効く指数（閾値ごと）

| 閾値(アトム) | 種別 | 指数 | 軽減 |
|---|---|---|---|
| 1（常時） | バフ | 特異点回数アトム倍率 | — |
| 1（常時） | バフ | ウィルスの昇華（達成 1.02 / 未達成 0.02） | — |
| 1e6 | バフ | 遺物69 | — |
| 1e10 | ペナ | ^0.5 | RN128（`0.5 + 0.05×Lv`、Lv10で1＝無効） |
| 1e50 | ペナ | ^0.833 | RN126（無効化） |
| 1.4e154 | ペナ | ^0.847 | RN127（無効化） |
| 1.6e256 | ペナ | ^0.714 | — |
| 1.8e308 | ペナ | ^0.12 | ツリー3.1 |
| 1e500 | ペナ | ^0.05 | ツリー3.2 |
| 1e1000 | ペナ | ^0.033 | ツリー6.2 |
| 1e2000 | ペナ | ^0.5 | — |
| 1e3000 | ペナ | ^0.333 | — |
| 1e5000 | ペナ | ^0.2 | — |
| 1e10000 | ペナ | ^0.1 | — |

サクリファイスダスト・拡散力系の強化（1e32/1e128/1e200/1e400/1e600）は原子獲得量に直接効かないため計算から除外しています。

### 区間積分
`E` は閾値を超えるたびに変わるので、区間 `[A0, A1]` を閾値で分割し、各区間で `E` 一定として合計します:

```
t = Σ ( 区間の桁幅 ) / ( E_区間 · log10(B) )
```

### ツリー弱化（仮の式）
ツリー3.1/3.2/6.2 は「/x」表記の値 `x`（1〜約1.17、10Lvごとに増分が 1/n! で減衰）で弱化します。実測から:

```
実効指数 = base / (2 − x)      base = 0.12 / 0.05 / 0.033（上限1）
```

> ⚠️ 正式な計算式は非公開のため近似です。正確に合わせたい場合は、ゲーム表示の実効指数を各ツリーの「ペナルティ」欄に直接入力してください。

### 必要アトム量（特異点N 到達）
wiki の区間式（1つ前の回数基準）に基づく近似:

| N | 必要アトム(log10) |
|---|---|
| 1〜10 | 1.80e308 |
| 11〜21 | `20(N−1)+120` |
| 22〜51 | `16(N−1)+200` |
| 52〜71 | `10(N−1)+500` |
| 72〜 | `(N−1)(N−71)+1200` |

（例: N=16→1e420、N=20→1e500、N=32→1e696、N=64→1e1130、N=72→1e1271）

### ローカル速度
ローカル速度 `s` のとき実時間は `1/s` になるため、算出した所要時間を `s` で割ります。

## 使い方

1. **特異点倍率**（特異点タブの倍率）と必要なら**現在のアトム量 / ローカル速度**を入力
2. **ペナルティ軽減**（RN126/127/128・ツリー3.1/3.2/6.2）と**バフ**（ウィルスの昇華・特異点回数アトム倍率・遺物69）を設定
3. **目標**を「特異点の回数」または「アトム量」で指定
4. 推定時間・実効レート・ペナルティ区間ごとの内訳が表示されます

各設定は自動保存されます。「設定と結果をコピー」でデバッグ用の全情報を書き出せます。

## 前提・制限

- ベース獲得量（特異点倍率・元素等）は周回中一定と仮定
- サクリファイスダスト・拡散力系の強化は未反映
- 必要アトム量は wiki 記載の区間式に基づく近似値
- **ツリー弱化の計算式は不明なため仮の式**。正確に合わせたい場合は、ゲーム表示の実効指数を各ツリーの「ペナルティ」欄に直接入力してください
